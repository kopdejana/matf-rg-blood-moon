# Keyboard instructions:

`W` / `S` / `A` / `D` - camera movement <br>
`LEFT` / `RIGHT` / `UP` / `DOWN` - camera rotation <br>
`B` - blood moon on/off <br>
`ESC` - exit

# Implemented
- Required: <br>
  - Weeks 1-8 <br>
  - Blending (discard) <br>
  - Face culling <br>
  - Advanced lighting <br>
- Group A: <br>
  - Cubemaps <br>
- Group B: <br>
  - HDR, Bloom <br>
